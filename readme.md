## 使用     说明

- 1. 修改 content 和 content-en 中的文件
- 2. `pelican content -s pelicanconf.py` 生成中文页面
- 3. `pelican content-en -s pelicanconf-en.py` 生成英文页面
- 4. `pelican -r -l` 本地预览
- 5. commit 提交更改，并 push 到 github
